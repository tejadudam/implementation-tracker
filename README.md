# 🚀 Implementation Tracker App

A Streamlit-based internal tool for tracking client implementations, tasks, and updates.

## Features
- Client Management
- Task & Subtask Tracking
- MOM & Requirements Logging
- Role-Based Access Control (Admin / Associate / Viewer)
- Feature Enablement Tracking
- Real-time Dashboard Insights

## Tech Stack
- Python
- Streamlit
- SQLite
- bcrypt (Authentication)

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
